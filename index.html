Community Coaching Series

AI for testing community coaching series of 15 collaborative hours of discovery learning by doing, facilitated by Maaret Pyhäjärvi.

Test the Potion Shop Application

About This Project

This collaborative learning experience runs through March 2026, organized by Aryadevi and Maaret. This particular project includes Christine Pinto's Potion Shop as test target for contemporary exploratory testing with AI as external imagination and tool for task expansion. The Potion Shop is a vibe coded application with intentional bugs, and provides a target rich exploration environment. Through task expansion of *fixing while reporting*, we can approach a more realistic simulation of testing while keeping the conceptual frame manageable.

The community group is limited to 10 participants, and is free of charge.

Contemporary Exploratory Testing Ideas

  • AI as external imagination.
  • AI as task expansion.
  • Results (actionable insights) are more important than intermediate artifacts.
  • Artifacts are an output of testing, to support future with knowledge captured at time we know the best.
  • Executable specifications of intent are an asset that invites us to explore.

Terminology

LLM (Large Language Model)
A type of artificial intelligence model trained on vast amounts of text data to understand and generate human-like text. Examples include GPT, Claude, and other transformer-based models that can assist with testing tasks, code analysis, and documentation.
Agent
An LLM agent runs tools in a loop to achieve a goal.
CLI (Command Line Interface)
A text-based interface that allows for a concise way for an agent to use a tool with a connected skill. Tends to save tokens, which translates to cost.
MCP (Model Context Protocol)
A standardized protocol that enables AI models to securely connect to and interact with external data sources, tools, and services. MCP expands an AI's capabilities beyond its training data by providing real-time access to relevant information and functionality.
Context
Limited window of what the model can consume.
RAG (Retrieval-Augmented Generation)
A context engineering approach that enhances AI model responses by first retrieving relevant information from external knowledge sources (databases, documents, APIs) and then using that retrieved context to generate more accurate, up-to-date, and contextually relevant answers. RAG allows models to access information beyond their training data without requiring full retraining.
Prompt
Instructions, queries, or input text provided to an AI model to elicit specific responses or behaviors. Effective prompting is crucial for getting AI to perform testing tasks accurately, whether generating test cases, analyzing bugs, or suggesting improvements.
Agent Files
Configuration files (such as .agent.md, .instructions.md, or .prompt.md) that define AI agent behavior, capabilities, and specialized workflows. These files contain instructions that customize how an AI agent operates within specific domains or for particular tasks, enabling consistent and targeted assistance.
Skills Files
Specialized knowledge files (typically SKILL.md) that package domain-specific expertise and detailed instructions for AI agents. Skills files contain comprehensive guidance for specific tasks or domains, allowing agents to access deep, contextual knowledge when working on particular types of problems or within specialized fields.
Frontier Models
State-of-the-art AI models that represent the cutting edge of artificial intelligence capabilities. These are typically the most advanced, largest, and most capable models available, often developed by leading AI research organizations. Examples include the latest versions of GPT, Claude, Gemini, and other models that push the boundaries of AI performance in reasoning, creativity, and problem-solving.
Local Models
AI models that run directly on your own hardware (computer, server, or device) rather than being accessed through cloud-based APIs. Local models provide benefits like data privacy, offline access, reduced latency, and cost control, but typically require more computational resources and may have lower capabilities compared to frontier cloud models.
Hugging Face
A comprehensive platform and ecosystem for machine learning, best known as the "GitHub of AI models." Hugging Face provides a hub for discovering, sharing, and using thousands of pre-trained models, datasets, and applications. It offers tools, libraries (like Transformers), and infrastructure that make AI models more accessible to developers and researchers, supporting both local deployment and cloud-based inference.
